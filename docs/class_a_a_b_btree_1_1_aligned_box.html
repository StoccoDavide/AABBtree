Detailed Description

template<typename Real, Integer N>
class AABBtree::AlignedBox< Real, N >

This class represents an axis-aligned box as a pair of opposite corners, i.e., a minimal corner and a maximal corner.

Template Parameters
Realthe type of the scalar coefficients
NSimension of the ambient space.
